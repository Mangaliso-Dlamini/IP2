Source

Uses up to the minute data sourced from Adzuna. Adzuna is a job advertisement search engine operating in 20 countries around the world. It aggregates job data from several thousand sources.

The data contained in this API is accessible through nine endpoints. It includes data about job advertisements, employment data(salary history, average salary, top companies, etc) and job categories. This web application visualises the employment data based on the regions and job categories available in each of the 20 countries

API Access

Data is imported from Adzuna API server using the jQuery specialised shortcut method $.getJSON() which harnesses the AJAX techniques. Several $.getJSON calls are made to various endpoints first to load locations and job categories, and ultimately to load employment data

How It Works

On load the page shows a dropdown selector where the can select a county to query its employment data. Users can select from the 20 countries in which Adzuna operates in. Once the user selects the country the two more dropdown selectors are loaded: one with the available regions and the other with available job categories in the selected country. The user can then select a region and job category then click the view button to view top hiring companies, salary distribution and salary history for the selected job category in the selected region. The top 5 hiring companies are present in a table format, while salary distribution and salary history are shown on interactive bar and line charts respectively.
